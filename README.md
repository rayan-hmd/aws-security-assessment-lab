# AWS Security Assessment Lab

## Project Overview

This project demonstrates the assessment and remediation of common AWS security misconfigurations using guidance from the ASD Information Security Manual (ISM) and Essential Eight Maturity Model, with remediations implemented using AWS-native security controls.

## Objective

The objectives of this project are to:

- Build an intentionally misconfigured AWS environment containing common security weaknesses.
- Demonstrate how common AWS misconfigurations can introduce security vulnerabilities and increase organisational risk.
- Assess the environment against relevant ASD Information Security Manual (ISM) and Essential Eight Maturity Model guidelines.
- Implement AWS-native security controls to remediate identified risks.
- Validate that remediations have been successfully applied.
- Develop practical skills in cloud security assessment, risk analysis, and remediation.
